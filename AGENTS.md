# AGENTS.md

## Purpose
This repository is structured so that strategic long-horizon thinking lives in an isolated module.

## Global routing rule
Default to the current product / code / business execution context.

Only route into `projects/future-thesis-lab/` when the task is about:
- future thesis;
- frontier research;
- long-range product bets;
- signal tracking;
- monthly / quarterly strategic review;
- whether a new direction fits the canonical long-term mission.

## Hard rule
Do **not** inject the full future-thesis module into ordinary daily execution tasks.

## Allowed always-visible summary
A short summary of the thesis can be referenced globally, but the full module is loaded on demand.

## Why
This preserves focus, avoids strategic noise, and keeps the thesis alive without letting it hijack the main execution loop.
