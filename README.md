# GPT-agent

This repository contains an isolated strategic module for long-horizon product thinking and frontier thesis work.

## Main principle
- **90%** focus stays on the current clinic AI business and execution.
- **10%** focus goes to a structured long-term thesis lab.

## Repository layout
- `projects/future-thesis-lab/` — strategic module for future thesis, signals, reviews, watchlists, ADRs, and roadmaps.

## Operating rule
This module is **separate by design**. It should be used frequently for strategic work, but it must **not** pollute the day-to-day execution loop of the main agent.

## Recommended use
Use the module when:
- evaluating a new major direction;
- reviewing frontier AI / healthcare shifts;
- running monthly or quarterly strategic reviews;
- deciding whether a signal deserves promotion into a watchlist, ADR, or roadmap.

Do not load the full module into every operational session.
